List active shortcodes on post edit page with drag and drop to post. Includes a few simple shortcodes.

== Description ==
Lists all active shortcodes below the post edit page. Shortcodes are easily dragged onto the post.

Includes a few simple shortcodes with usage instructions that I wrote for other projects. 

Grab Bag shortcodes:
 RSS feed
 List of links to pages, Posts, or Custom Post Types
 Wikipedia link
 Amazon affiliate search link
 PayPal add to cart
 PayPal view cart. 

Any ideas or suggestions for more simple shortcodes to add to the plugin are welcomed.
 
== Installation ==
1. Download the plugin.
2. Upload the plugin to your wp-content/plugins directory.
3. Activate the plugin.
There will be a list of shortcodes on the post/page edit page.
